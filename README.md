#### 一款模拟资源管理器的摸鱼桌面看股软件（实时行情）

![](http://qiniu.hgteam.cn/ac.png)

- 进程 (自定义的名称)
- CPU (涨跌幅)
- 磁盘 (价格)
- 网络 (涨跌数)

#### 代码运行
需要安装node.js环境

clone项目代码

```
npm install
npm start
```


### 添加自选股
快捷键 'alt' 打开命令输入框

`add 进程名称 股票代码`

> 比如添加贵州茅台，`add MT SH600519`

进程名称 是自定义的，可中文可英文

股票代码要带上SH或SZ

### 删除自选股

`remove 进程名称`

### 清空自选股

`clear`
